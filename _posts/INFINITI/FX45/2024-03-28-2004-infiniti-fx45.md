**2004 INFINITI FX45**

The 2004 INFINITI FX45 is a luxury SUV with a powerful 4.5-liter V8 engine making 315 horsepower and 334 lb-ft of torque. The FX45 is also equipped with a 5-speed automatic transmission and rear-wheel drive or optional all-wheel drive. It has a spacious cabin, a comfortable ride, and excellent acceleration. However, it has some known issues that owners should be aware of.

**Engine**

* **VK45DE 4.5L V8**
    * **Exhaust camshaft sensor failure:** The exhaust camshaft sensor can fail, causing the engine to run rough and stall. The sensor needs to be replaced. The cost to fix this issue is $200 - $400.
    * **Rear main seal leak:** The rear main seal can leak, causing oil to leak out of the engine. The seal needs to be replaced. The cost to fix this issue is $1,000 - $2,000+.
* **V35DE 3.5L V6**
    * **Timing chain tensioner failure:** The timing chain tensioner can fail, causing the timing chain to jump or break. This can lead to catastrophic engine failure. The tensioner and chain need to be replaced. The cost to fix this issue is $2,500+.

**Transmission**

* **5-speed automatic**
    * **Torque converter shudder:** The torque converter can shudder when the transmission is shifting, causing a vibration in the vehicle. The torque converter needs to be replaced. The cost to fix this issue is $1,000 - $2,000.

**Suspension**

* **Rear differential mount failure:** The rear differential mount can fail, causing the differential to move around and make noise. The mount needs to be replaced. The cost to fix this issue is $500 - $1,000.
* **Front lower control arm bushings wear:** The front lower control arm bushings can wear out, causing the suspension to be noisy and loose. The bushings need to be replaced. The cost to fix this issue is $200 - $400.

**Electrical**

* **Battery drain:** The battery can drain prematurely, especially when the vehicle is not being driven for long periods of time. The parasitic drain needs to be found and fixed. The cost to fix this issue varies depending on the source of the drain.
* **Alternator failure:** The alternator can fail, causing the battery to not be charged and the vehicle to stall. The alternator needs to be replaced. The cost to fix this issue is $500 - $1,000.

**Exterior**

* **Headlight condensation:** The headlights can develop condensation inside them, causing them to be foggy and less effective. The headlights need to be replaced. The cost to fix this issue is $500 - $1,000.

**Interior**

* **Heater core failure:** The heater core can fail, causing coolant to leak into the cabin and create a musty smell. The heater core needs to be replaced. The cost to fix this issue is $1,000 - $2,000+.
* **Power window switch failure:** The power window switches can fail, causing the windows to not operate properly. The switches need to be replaced. The cost to fix this issue is $200 - $400.

**Recalls**

* NHTSA Campaign Number: 14V371000   - Hazard Light Switch Failure
* NHTSA Campaign Number: 14V368000   - Backup Camera System Failure
* NHTSA Campaign Number: 13V337000   - Fuel Cap Assembly
* NHTSA Campaign Number: 07V396000   - Child Seat Tether Anchors

**Summary**

The 2004 INFINITI FX45 is a reliable and capable SUV, but it does have some known issues that owners should be aware of. The severity and cost of the issues vary, but some of them can be quite expensive to fix. If you are considering purchasing a used FX45, it is important to have it inspected by a qualified mechanic to ensure that it does not have any major problems.
